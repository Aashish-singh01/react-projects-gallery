## 🚀 Run Locally

```bash
# Clone the repo
git clone https://github.com/yourusername/react-projects-gallery

# Go into the folder
cd react-projects-gallery

# Install dependencies
npm install

# Start dev server
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.
